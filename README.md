# Homebrew Tap for CoinGecko CLI

Homebrew formulae for [CoinGecko CLI (`cg`)](https://github.com/coingecko/coingecko-cli).

## Install

```sh
brew install coingecko/coingecko-cli/cg
```

Or tap first, then install:

```sh
brew tap coingecko/coingecko-cli
brew install cg
```

## More info

See the [main repository](https://github.com/coingecko/coingecko-cli) for documentation and usage.
